Interactive Elements

In-text link

CSS selector:

.intext-link

Sample code:

<p><a href="#" class="intex-link">in-text link</a></p>

Rendered element:

in-text link

Text Input Field with Label

CSS selector:

.input-label

Sample code:

<label class="input-label"></label><input class="input-box" type="text">

Rendered element:

Bulleted Lists

CSS selector:

.bulleted

Sample code:

<ul class="bulleted"></li>Bulleted List 1</li> </li>Bulleted List 2</li> </li>Bulleted List 3</li></ul>

Rendered element:

  • Bulleted List 1
  • Bulleted List 2
  • Bulleted List 3

Numbered Lists

CSS selector:

.list-num

Sample code:

<ol class="list-num"></li>Number List 1</li> </li>Number List 2</li> </li>Number List 3</li></ol>

Rendered element:

  1. Number List 1
  2. Number List 2
  3. Number List 3
